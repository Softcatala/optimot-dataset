# Optimot Dataset

The **Optimot dataset** consists of **4,011 entries** extracted from [Optimot datasource](https://aplicacions.llengua.gencat.cat/llc/AppJava/index.html).

Each entry addresses a common question about the **Catalan language**, offering explanations and possible solutions for each case.

**File:** [optimot.jsonl](optimot.jsonl) (JSON Lines)

**Date:** 2026/7/8 (dataset download date; each entry includes its source update date in `Darrera versió`)

**Fields:** `Fitxa`, `Darrera versió`, `Títol`, `Resposta`, `source`, `source_id`, `source_url`, `download_date`.
